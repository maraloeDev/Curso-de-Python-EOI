# Curso-de-Python-EOI
Curso de Python – EOI (6 semanas) Este repositorio recopila los ejercicios y proyectos realizados durante el curso intensivo de Python en EOI. Contiene prácticas sobre sintaxis básica, estructuras de datos (listas, diccionarios, tuplas), control de flujo, funciones, módulos, manejo de errores y fundamentos de POO.
